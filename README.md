# 1-Projeto---HTML-CSS---Convertedor-Moedas

Convertedor de Moedas
Este é o meu primeiro projeto prático desenvolvido durante as aulas introdutórias de HTML e CSS. O objetivo foi criar a interface de um conversor de moedas moderno e responsivo.

* Tecnologias Utilizadas
HTML5: Estruturação dos campos de entrada (input), botões e imagens.

CSS3: Estilização visual, incluindo:

Uso de cores personalizadas e contrastantes.

Bordas arredondadas para um visual moderno (border-radius).

Centralização de elementos utilizando margens automáticas.

* O que aprendi neste projeto
Como organizar elementos dentro de uma div.

Diferença entre estados de inputs, como o uso do atributo readonly para resultados.

Importância do reset básico de margens e preenchimentos (* { margin: 0; padding: 0; }).

* Próximos Passos
Aplicar Flexbox para melhorar o alinhamento dos elementos.

Implementar a lógica de conversão real utilizando JavaScript.

Adicionar consumo de uma API de cotações em tempo real.
